# 📊 Sales Analysis in R

## 📌 Project Overview
This project analyzes sales data using R to uncover trends, customer behavior, and revenue patterns.

## 📁 Project Structure
- data/ → raw datasets  
- visuals/ → generated charts and plots  
- analysis.R → main analysis script  
- README.md → project documentation  

## 🔧 Tools & Libraries
- R
- dplyr
- ggplot2
- tidyr
- lubridate
- ggrepel

## 📈 Key Analysis
- Monthly and yearly sales trends
- Sales by category and region
- Peak and dip month identification
- Top customers analysis
- Customer retention segmentation
- Revenue contribution by customer type

## 💡 Key Insights
- Sales show strong seasonal patterns
- Technology category drives highest revenue
- Repeat customers contribute significantly to revenue
- Customer retention is a key growth opportunity

## 🚀 How to Run
1. Open `analysis.R` in RStudio  
2. Ensure required packages are installed  
3. Run the script step by step
